⚡ CRATEVAULT — Firebase Setup

To play, you need a free Firebase project. Follow these steps once, then paste your config below.

Step 1: Go to console.firebase.google.com → Add project → name it anything.
Step 2: In the project, click Firestore Database → Create database → Start in test mode.
Step 3: Click Authentication → Get started → Enable Email/Password sign-in.
Step 4: Click the ⚙️ gear → Project settings → scroll down to "Your apps" → click </> Web → register app → copy the firebaseConfig values below.

Your config is saved locally in your browser — never shared with anyone.
